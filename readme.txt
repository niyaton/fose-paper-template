タイトルと著者欄が2行に渡った際にレイアウトが崩れる件の対処方法について
------------------------------------------------------------------------
compsoft.sty
デフォルトでは以下のとおりですが、タイトルと著者欄の*両方*が2行に渡る際、
859行目先頭の%を削除し、861行目の先頭に%を追加ください。

858: %rePiT2017　タイトルと著者がいずれも2行に渡る場合，以下の行をご使用下さい．
859: %\hbox{\vbox to233.5\Q{\baselineskip=10mm
860: %上記の場合を除き，以下がデフォルトとなっています．
861: \hbox{\vbox to193.5\Q{\baselineskip=10mm % 193.5\Q = 9*\baselineskip
